# Gerador-de-senha
 Projeito feito com as tecnologias HTML5, CSS3 e JavaScript
 
 Podendo selecionar de 5 a 25 caracters e gerando as senhas aletoriamente e também com a função de copiar a senha ao clicar em cima.
